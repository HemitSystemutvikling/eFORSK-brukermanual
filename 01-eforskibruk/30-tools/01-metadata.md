# Metadata

Under **Skjematype metadata** kan man laste ned metadata for valgte skjematyper i prosjektet sitt. Metadata i eFORSK beskriver innholdet og strukturen i en database, for eksempel informasjon om variabler, skjematyper, spørsmål og svaralternativer. Dette gjør det enklere å forstå, gjenfinne og bruke data på tvers av studier. God bruk av metadata bidrar til kvalitet, sporbarhet og effektiv håndtering av data gjennom hele forskningsløpet og kan være krav om å arkivere etter prosjektslutt.
